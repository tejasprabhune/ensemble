# World API

This page is the reference for authoring a world plugin. It covers
the Python surface a world's `__init__.py` reaches for, the
`world.toml` manifest that ties the world to its rust crate and
personas directory, and the Rust `WorldState` trait that worlds with
a typed rust core implement.
