# Tools

This page documents how to declare a tool a world's agents can
call. It covers the Python `tool()` helper and the underlying
`PluginTool` shape, the JSON-string ABI the runtime expects, and
the four optional capabilities (timeouts, resource locks, progress
emission, cost annotations) along with sandbox-mode dispatch.
