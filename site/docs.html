writing a world

A world is a Rust struct that implements the WorldState trait. The trait has three associated types and three methods:

pub trait WorldState: Send + Sync + 'static {
    type ToolCall:  DeserializeOwned + Send;
    type ToolEffect: Serialize + Send;
    type Diff:       Serialize + Send;

    fn apply(&mut self, call: Self::ToolCall)
        -> Result<(Self::ToolEffect, Self::Diff), ToolError>;

    fn snapshot(&self) -> Vec<u8>;
    fn restore(&mut self, snapshot: &[u8]) -> Result<(), RestoreError>;
}

The Diff is what the trace viewer renders in the right-hand panel. Make it rich enough to reconstruct what changed (table, row id, field, old, new).

registering tools

Tools are Rust closures the agent can call by name. Each tool has a JSON Schema for its parameters and a closure that maps args to an effect.

tools.register(Tool::new(
    "lookup_user",
    "Look up a user by id.",
    json!({"type":"object","properties":{"user_id":{"type":"string"}}}),
    |args| {
        let user_id = args["user_id"].as_str().unwrap();
        let rec = state.lock().lookup_user(user_id)?;
        Ok(json!({"ok": true, "data": rec}))
    },
));

scaffold a new world

ensemble init my_world
cd my_world
cargo build
ensemble run my_world.smoke

trace format

Traces are JSONL. One line per event; events are ordered by tick.

{
  "tick": 7,
  "ts_ms": 1716000123,
  "actor": "rep1",
  "message_id": "msg-...",
  "payload": {
    "kind": "tool_call",
    "name": "issue_refund",
    "args": {"user_id": "u-alice", "amount_cents": 15000, "reason": "..."}
  }
}

Event kinds: user_message, agent_message, tool_call, tool_result, state_diff, system.

hidden state and persona prompts

Personas (prompted or trained) hold hidden state that is injected into the system prompt inside an unambiguous tag. Non-revelation is a training problem in v0; see the personas page.

known limits (v0)

  • No streaming LLM responses.
  • One trainer (TRL + DPO + LoRA); no verl, no prime-rl.
  • No hosted platform: everything is local + Modal/SkyPilot.
  • Persona consistency evaluator is not implemented (page is a placeholder).
